# Augusta Defended

### Golf Data Storytelling & Static Analytics Site

Augusta Defended is a Vite/React/TypeScript data-storytelling site that turns static tournament JSON artifacts into interactive analytics pages for a Masters repeat-champion narrative.

## Problem

Tournament coverage often collapses to a leaderboard. This project builds a more navigable analytical surface: scorecards, player profiles, field and top-10 comparisons, repeat champion benchmarks, cumulative paths, and round-by-round context.

## System

- React Router pages for home, profile, and repeat context.
- Static JSON artifact imports.
- TypeScript selectors and hooks for chart-ready data transforms.
- Node/Playwright pipeline for fetching and packaging tournament data.
- Scorecard and course-data validation.
- Recharts visualizations and deterministic data packaging.

## Stack

Vite, React, TypeScript, Recharts, Node, Playwright, JSON artifacts, Vercel.
