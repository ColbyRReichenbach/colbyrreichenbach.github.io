# MLB Voting Visualized

## Project Overview
An interactive data storytelling project visualizing **20 years of MLB Award Voting trends**. This project showcases the ability to turn complex historical datasets into intuitive, explorable narratives.

## Key Highlights
*   **Interactive Storytelling:** deeply interactive dashboards that allow users to explore voting patterns for MVP and Cy Young awards across two decades.
*   **Complex Data Modeling:** Aggregated and cleaned inconsistent historical voting data to create a unified schema for analysis.
*   **Visual Design:** Applied best practices in data visualization to highlight outliers, trends, and voting biases clearly.
*   **Public Access:** Deployed on **Tableau Public** for widespread accessibility and interaction.

## Tech Stack
*   **Tableau**: Advanced data visualization and dashboarding.
*   **Data Cleaning**: Excel/SQL for data preparation.
*   **Visual Analytics**: Trend lines, heat maps, and interactive filters.
