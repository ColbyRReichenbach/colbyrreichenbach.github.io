
# Pulse Tracker: Hybrid Athlete Training Platform

Pulse Tracker is a high-performance workout tracking and AI-coaching dashboard designed for hybrid athletes, integrating strength training, aerobic conditioning, and biometric analysis into a single ecosystem.

## Core Features
*   **Hybrid Training System**: A structured 52-week training methodology for concurrent strength and endurance development. Includes tracking for resistance training, Olympic lifting, and Zone 2 cardio.
*   **Intelligent AI Coaching**: A stateful, multi-turn AI assistant (ECHO-P1) powered by GPT-4o-mini that provides real-time modifications, injury management advice, and data analysis.
*   **Biometric & Performance Analytics**: Centralized visualization of HRV, sleep quality, and progressive overload trends.

## Tech Stack
*   **Frontend**: Next.js 16 (App Router), TypeScript, Vanilla CSS, Framer Motion
*   **Backend**: Supabase (PostgreSQL + RLS), Edge Functions
*   **AI**: OpenAI GPT-4o-mini, Vortex Optimization
*   **Infrastructure**: Redis (Upstash), Sentry Monitoring
