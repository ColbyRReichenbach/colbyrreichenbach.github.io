AppTrail

A full-stack workflow product that centralizes applications, jobs, contacts, emails, resumes, research signals, and AI copilot workflows across a FastAPI backend, React dashboard, and Chrome extension.

Repo All work

Problem

A serious job search is fragmented across ATS pages, Gmail, resumes, contacts, company research, interviews, follow-up timing, and repeated browser workflows. AppTrail treats that process as a structured operating system instead of a spreadsheet.

System

FastAPI backend with async SQLAlchemy/Alembic models, PostgreSQL, Redis/Celery scheduled jobs, Google/Gmail sync, React/Vite dashboard, Chrome Manifest V3 extension, AI orchestration services, model-call telemetry, consent checks, safety decisions, and admin governance surfaces.
